このリポジトリはローレンツモデルをさまざまな回帰モデルで推定するプログラムである。

[main.py](main.py)は自作のローレンツ方程式の推定プログラムである。
4次精度のRunge-Kutta法を副作用のない関数で定義し、ローレンツモデルをクラスで定義した。scikit-learnライブラリで線形回帰、Ridge回帰、LASSO回帰の3つでローレンツ方程式の係数を推定し、予測精度の違いを確認する。

[lorenz_solver.py](lorenz_solver.py)はGeminiによって生成されたローレンツ方程式の係数を推定するプログラムである。
自作のプログラムと同様に4次精度のRunge-Kutta法とローレンツモデルを採用している。
ローレンツ方程式の推定は必要な係数のみを推定するように最適化されている。
